# rudder-pi

Rudder-pi is a modular Raspberry Pi based boat control system.

The Raspberry Pi acts as the main boat computer:
- safe motor control with hardware failsafes
- local web status and control UI
- logging and watchdogs

An Android phone (USB tethered) provides:
- LTE connectivity
- GPS
- camera
- IMU sensors

The project is developed step-by-step with a strong focus on reliability and safety.
